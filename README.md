# Cooperacion de agentes llm 

Se implementa la investigacion del paper siguiente

**Cooperation Breakdown in LLM Agents Under Communication Delays Keita Nishimoto, Kimitaka Asatani and Ichiro Sakata The University of Tokyo keita-nishimoto@g.ecc.u-tokyo.ac.jp** 
 
Para la  ejecucion seguir los pasos
```bash
    bash limpieza.sh 
    bash ejecucion.sh

    bash ordenes.sh {i-carrito} velocidad-lineal direccion-angular-timon
    bash ordenes.sh 1 0.5 30
```
